{\footnotesize\ttfamily void Flight\+Manager\+::print\+Num\+Flights\+City (\begin{DoxyParamCaption}\item[{const std\+::string \&}]{city\+Name }\end{DoxyParamCaption})}



Print the number of flights that a city has. 

The number of flights is considered to be out-\/degree (Flights that have this \doxylink{class_airport}{Airport} as source)

plus it\textquotesingle{}s in-\/degree (Flights that have this \doxylink{class_airport}{Airport} as destination).

The Time Complexity should be O(1) however, if a city has more than one \doxylink{class_airport}{Airport} then Time Complexity

will be O(\+Number of Airports of that City). Also, due to the use of an hashmap, in the worst case

the time complexity can be O(n) or even O(n + Number of Airports of that City) where n is the

size of the hashmap.
