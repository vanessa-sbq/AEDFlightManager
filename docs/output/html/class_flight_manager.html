◆ printFlightOptionMinimalAirlines()

void FlightManager::printFlightOptionMinimalAirlines ( vector< Vertex< Airport * > * > source,
vector< Vertex< Airport * > * > dest )

Prints the flight from source to destination vertices using the least different airlines possible.

This function iterates through each combination of source and destination airports to find the path using the minimum number of different airlines, perfoming a series of operations on the graph (airportNetwork). First, resets the indegrees of each vertex ensuring accurate traversal by the ^bfsMinimalAirports^ function and then compares each path found to find the desired path.

Time Complexity: O(n^2 * V + E), where n is both the worst case size of source and destination vectors.

◆ printMaxTrip()

void FlightManager::printMaxTrip ( )

Prints the flights (source and destination) that have the most amount of stops.

Using a bfs approach we check for every single vertex what it's max depth is. In the case where the depth

is greater than the previous one we store the maximum depth as well as the source(s) and the destination(s) airports.

Time complexity: O(V * ( 2 * V + E)) where V is the number of vertexes (Airports) that the graph has and E

is the number of edges (flights) that an airport has.

void FlightManager::printNumDestinationsForGivenAirport ( const std::string & airportCode,
std::vector< int > filters )

Print the number of airports, countries or cities from a given airport.

From a given airport code we will check it's neighboring vertices while also keeping track of the countries, cities and airports that

were visited for those cases where there are two cities with the same name or a single city has many airports.

Time Complexity for this is O(V) + O(E)*O(n) (lookup for vertex + iterating over edges * number of cities (or countries) that were already visited.)

Print the number of flights that a city has.

We visit every airport of the city and add it's in-degree plus it's out-degree.

Time Complexity for this is (O(V) + O(E))*O(x) where V is the number of airports, E then number of edges

and x is the number of airports of the city.

void FlightManager::printNumFlightsOutOfAirport ( )

Prints the number flights out of an airport.

This function prompts the user for an airport code, searches for the corresponding vertex in the airport network, and then prints the number of flights and unique airlines departing from the specified airport.
