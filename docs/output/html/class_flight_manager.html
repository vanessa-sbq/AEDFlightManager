◆ printNumFlightsCity()

void FlightManager::printNumFlightsCity ( const std::string &  cityName)

Print the number of flights that a city has.

The number of flights is considered to be out-degree (Flights that have this Airport as source)

plus it's in-degree (Flights that have this Airport as destination).

The Time Complexity should be O(1) however, if a city has more than one Airport then Time Complexity

will be O(Number of Airports of that City). Also, due to the use of an hashmap, in the worst case

the time complexity can be O(n) or even O(n + Number of Airports of that City) where n is the

size of the hashmap.
