BFS algorithm to discover the shortest path with minimal different airlines.

Parameters
sourceStarting Vertex
targetTarget Vertex
Returns
Vector containing every path from source to target vertices with set and number of airlines used.

The BFS algorithm is employed to explore the graph, starting from the source airport vertex. The algorithm continues until all minimal paths to the target airport are reached. The indegree of each vertex is decremented during traversal to ensure that every edge is used. The resulting minimal paths are stored in a vector and returned.

The bfsPath structure is used to keep track of paths and the number of airlines used.

Time complexity: O(V + E) where V is the number of vertices and E the number of edges.

Print the number of flights that an airline has.

The number of flights is calculated using a dfs approach. We use the dfs algorithm to visit every airport

and check it's edges (Flight). From here we verify if the any flight of this airport is of the airline we are looking for.

If it is then we will increment the counter.

Time Complexity for this is O(V) + O(E) where V is the number of airports and E then number of edges.
